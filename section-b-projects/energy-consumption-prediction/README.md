\# Energy Consumption Prediction and Bill Reduction



Ibtisam rashid 70137366



\## Problem Statement

Predict household energy consumption and suggest bill-reducing recommendations using machine learning.



\## Dataset

Household Electric Power Consumption dataset (Kaggle): \[Link](https://www.kaggle.com/uciml/household-electric-power-consumption)



\## Model / Technique

\- Linear Regression for prediction

\- Confusion Matrix for evaluation

\- FastAPI backend + Next.js frontend



\## How to Run

1\. Clone the repo

2\. Install Python dependencies: `pip install -r requirements.txt`

3\. Start backend: `uvicorn backend.backend:app --reload`

4\. Start frontend: `npm install \&\& npm run dev`



\## Results

\- Predicted energy usage

\- Estimated monthly bill

\- Dynamic confusion matrix
